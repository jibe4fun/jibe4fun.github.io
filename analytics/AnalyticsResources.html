  • Other R Resources:

    R (http://www.r-project.org/) -- to download R
    R Studio (http://www.rstudio.com/) -- to download R Studio - a development environment for R, a must!!
    RSeek (http://www.rseek.org/) -- great search tool specific to R
    R Packages (http://cran.r.project.org/web/packages/) -- a list of all R packages (Comprehensive R Archive Network, CRAN)
    R Views (http://cran.r-project.org/web/views/) -- a view = group of packages for a specific function (e.g., graphics, econometrics)
    Data Camp (https://www.datacamp.com/) -- a great interactive site to learn R
    R Instructor (https://itunes.apple.com/us/app/r-instructor/id527985215?mt=8) -- an iPhone/iPad app with R resourcess
    Shiny R Studio Gallery (http://shiny.rstudio.com/gallery/) -- a great R Studio gallery for data visualization
    R for Everyone (www.amazon.com/dp/B00HFULELW/) -- great book to get started on statistics in R
